Kudu Quickstart

This quickstart shows how to set up and run a VirtualBox virtual machine pre-installed with Kudu, Kudu_Impala, and CDH. This is the quickest way to take Kudu for a spin.

Get The Kudu Quickstart VM

Follow these instructions to download and set up the Kudu Quickstart VM.

The VM contains Kudu, Impala, and other Hadoop ecosystem projects, for your convenience.
